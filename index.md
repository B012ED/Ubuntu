This application makes it easy to install ubuntu without root.

**how to install and usage:**

**Termux:**
* `pkg install php`
* `pkg install git`
* `git clone https://github.com/B012ED/ubuntu.git`
* `cd ubuntu`
* `php ubuntu.php`

**Linux:**
* `apt-get install php`
* `apt-get install git`
* `git clone https://github.com/B012ED/ubuntu.git`
* `cd ubuntu`
* `php ubuntu.php`
